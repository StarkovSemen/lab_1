# Задание 1

### Текст задачи

Сформировать список из чисел, обратных вводимым значениям.

### Алгоритм решения

1)Запрашиваем у пользователя кол-во чисел списка
2)Вызываем рекурсивную функцию
2.1)Проверка кол-во чисел в списке, если равно 0, то возвращаем перевернутый накопленный "список", иначе пункт 2.2
2.2)Запрашиваем элемент
2.3)Ввод эллемента
2.4)Переворачиваем число рекурсивной функцией (функция работает через деление числа на 10 и обрезкой последней цифры числа)
2.4)Уменьшаем кол-во чисел и записываем вводимое число в голову "списка", тем самым получается перевернутый список
2.5)Рекурсивно вызываем функцию, пока кол-во чисел не равно нулю 
3)Выводим наш список

## Задание 2

### Текст задачи

Найти количество чётных цифр в записи натурального числа.

### Алгоритм решения

1)Запрашиваем у пользователя натуральное число
2)Проверка, если вводимое число 0, то 1(так как 0 четное число), иначе вызываем рекурсивную функцию
3)Рекурсивная функция
3.1)Проверяем что число не равно 0
3.2)Если не равно 0, то переходим к шагу в 3.3
3.3)Записываем последнию цифру числа
3.4)Записываем обрезаное число
3.5)Если последняя цифра числа при деление на 0 дает остаток 0, то прибавляем 1 к "счетчику четных чисел" и вызываем рекурсивно функцию
3.6)Иначе вызываем рекурсивно функцию
4)Выводим кол-во чертных цифр в числе
